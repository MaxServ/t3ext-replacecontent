================
What does it do?
================

It searches for patterns in the generated TYPO3 page using one or more regular expressions and then replaces those
instances with a replacement.

The search and replace takes place after the complete page has been generated by TYPO3.

It uses the tslib_fe hook for post-processing of page content cached/non-cached: :code:`contentPostProc-all`.
